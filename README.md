# HackRune

HackRune is a lightweight OSINT (Open Source Intelligence) toolkit that automates data collection and analysis. It combines shell scripts, a Node.js backend, and Google's Gemini AI to gather information and generate intelligent insights.

## Features

* 🔍 Automated OSINT data collection
* 🤖 AI-powered analysis using Gemini
* 🌐 Simple web interface
* ⚡ Fast and lightweight setup
* 📊 Easy-to-read analysis results

## Tech Stack

* Node.js
* JavaScript
* Shell Script (Bash)
* Google Gemini API
* HTML/CSS

## Installation

```bash
git clone https://github.com/gurtej9602/HackRune.git
cd HackRune
npm install
```

## Run

```bash
npm start
```

Then open the application in your browser.
